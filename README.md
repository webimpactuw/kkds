# KKDS Website

## Description
This is a website for Kalamandapam Kuchipudi Dance School (KKDS). It covers the basic pages required, including a home page, contact page, and events page. This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
